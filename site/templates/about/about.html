LITMUS ONLINE JUDGE

LITMUS : JBNU Online Judge from ALPS@JBNU

Litmus는 ALPS에서 제작한 Online Judge Platform 이며, 오픈소스 기반으로 제작되었습니다.
교과목 실습 및 대회 운영에 활용 가능하며, 여러 언어 옵션을 활용하여 문제를 해결할 수 있습니다.
Litmus is an Online Judge Platform by ALPS@JBNU, based on Open-Source(DMOJ).
The platform can be utilized for coursework labs and competitions, and can utilize multiple language options to solve problems.

보안 관련

  • 자동 로그아웃: 1시간 후 자동 로그아웃 되도록 수정했습니다.
  • 문제 암호화: 문제 데이터에 대한 암호화 기능을 추가했습니다.

문제 및 테스트 케이스

  • 대회 생성 문제 선택: 대회 생성 시 문제 선택에 파일 체크박스 기능을 추가했습니다.
  • 언어별 시간 제한: 각 프로그래밍 언어별로 시간 제한을 개별 설정할 수 있도록 기능을 추가했습니다.
  • 테스트 케이스 파일 제한 완화: 업로드 가능한 테스트 케이스 파일의 크기 제한을 완화했습니다.
  • 압축 파일 업로드 자동 설정: 테스트 케이스 압축 파일 업로드 시 자동으로 테스트 케이스가 설정되도록 개선했습니다.
  • YAML 파일 동기화 오류 수정: 문제와 YAML 파일 간의 동기화 오류를 해결했습니다.
  • 교수자 테스트 케이스 확인: 교수자 계정에서 모든 문제의 테스트 케이스를 확인할 수 있도록 개선했습니다.

대회 / 과제

  • 문제 풀이 현황 확인: 대회/과제의 문제 목록에서 풀이 현황을 바로 확인할 수 있도록 수정했습니다.
  • 제출 코드 확인: 학생 사용자가 대회/과제 종료 후에도 제출한 코드를 조회할 수 있도록 수정했습니다.
  • 리스트 정렬: 종료된 대회/과제 리스트에서 로그인한 사용자가 생성한 항목이 상단에 표시되도록 수정했습니다.
  • 정답률 및 정답자 수 계산: 대회/과제 문제의 정답률 및 정답자 수가 해당 대회/과제 내에서만 산출되도록 수정했습니다.
  • 랭킹 페이지 개선: 랭킹 페이지의 디자인 및 일부 기능을 개선했습니다.

기타

  • STATUS 탭 추가: 채점 현황 및 시스템 상태를 확인할 수 있는 STATUS 탭을 추가했습니다.
  • 사용자 매뉴얼: 사용자 매뉴얼을 JHelper에 추가했습니다.
  • 댓글 창 크기 제한: 댓글 입력창의 크기 제한을 설정했습니다.
  • 캘린더 버튼 색상: 관리자 페이지 캘린더 이전/다음 버튼 색상을 수정했습니다.

Litmus 사용 안내

Litmus는 PC 환경에 최적화되어 있으며, 모바일에서는 정상적인 사용이 어려울 수 있습니다. 따라서, 원활한 Litmus 사용을 위해 PC 환경을 권장합니다.
Litmus는 Google Chrome 브라우저의 사용을 권장하며, 이외 브라우저에서는 오류가 발생할 수 있습니다.
Litmus is optimized for a PC environment, and may not work properly on mobile. Therefore, we recommend a PC environment for a smooth experience with Litmus.
Litmus is more compatible with the Google Chrome browser than any other browser, and may cause errors in other browsers.
